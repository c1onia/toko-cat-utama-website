# Toko Cat Utama Website

Responsive company website for Toko Cat Utama built with Next.js, TypeScript, Tailwind CSS, and reusable React components.

## Run locally

```bash
pnpm install --store-dir ./work/.pnpm-store
pnpm dev
```

## Validate

```bash
pnpm typecheck
pnpm build
```

## Brand assets

The uploaded logo, mascot, hero facade, and color-bar assets are served from `public/brand`.

Contact links are centralized in `lib/site-data.ts`.
